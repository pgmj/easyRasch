# Plot with infit ZSTD and item location ZSTD is sample size sensitive, see "RIitemfitPCM" for options

Plot with infit ZSTD and item location ZSTD is sample size sensitive,
see "RIitemfitPCM" for options

## Usage

``` r
RIinfitLoc(dfin, samplesize, nsamples)
```

## Arguments

- dfin:

  Dataframe with item data only

- samplesize:

  Desired sample size in multisampling (recommended 250-500)

- nsamples:

  Desired number of samples (recommended range 10-30)
