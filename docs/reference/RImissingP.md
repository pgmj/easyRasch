# Creates a figure with item missing data descriptives for participants

The y axis is ordered by most missing responses.

## Usage

``` r
RImissingP(data, output, N = 10)
```

## Arguments

- data:

  Dataframe/tibble to create table from

- output:

  Optional dataframe with participants with missing data

- N:

  Show n participants with most missing data (default = 10)

## Details

Sample use: `RImissingP(data)`
