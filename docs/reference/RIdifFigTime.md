# Create a DIF line graph over time, showing groups' PCM item locations

This function is specifically intended for examining DIF over time, to
create a figure that follows each item location on the y axis with time
on the x axis.

## Usage

``` r
RIdifFigTime(dfin, dif.var)
```

## Arguments

- dfin:

  Dataframe with item data only

- dif.var:

  DIF variable
