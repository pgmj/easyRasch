# Plot with outfit ZSTD and item location ZSTD is sample size sensitive, see "RIitemfitPCM"

Plot with outfit ZSTD and item location ZSTD is sample size sensitive,
see "RIitemfitPCM"

## Usage

``` r
RIoutfitLoc(dfin, samplesize, nsamples)
```

## Arguments

- dfin:

  Dataframe with item data only

- samplesize:

  Desired sample size in multisampling

- nsamples:

  Desired number of samples (recommended range 10-50)
