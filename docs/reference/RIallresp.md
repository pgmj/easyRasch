# Create table with summarized responses across all items.

Create table with summarized responses across all items.

## Usage

``` r
RIallresp(dfin, pdf.out, fontsize = 15)
```

## Arguments

- dfin:

  Dataframe with item data only

- pdf.out:

  Set to TRUE to get PDF-compatible table (kableExtra)

- fontsize:

  Set font size for PDF-compatible table
